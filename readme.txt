Keyboard shortcuts:

Cursor keys - move

P - load level data from clipboard
T - start/stop recording test (output in console)
TAB - toggle editor mode
EDITOR MODE CONTROLS:
    1 - place dog
    2/3 - place people
    4 - place cushion (place cushion on itself to remove it)
    Q/E lower/raise tile
    WSAD - create ramp
    IJKL - extend map border NWSE 
    SHIFT+IJKL - contract map border NWSE
